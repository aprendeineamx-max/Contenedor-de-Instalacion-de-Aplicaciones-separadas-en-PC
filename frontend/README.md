# Frontend (Panel Web)

Aplicación Next.js 14 + TypeScript orientada a:
- Crear contenedores mediante un wizard guiado.
- Visualizar estado, métricas y logs en tiempo real (TanStack Query + WebSockets).
- Ejecutar acciones (instalar, snapshot, exportar, ejecutar scripts) de forma remota.

Stack previsto: Next.js (App Router), TailwindCSS, shadcn/ui, Zustand, Recharts.

## Próximos Pasos

- Consumir APIs simuladas hasta que el backend esté disponible.
- Definir componentes compartidos (tablas, tarjetas de estado, formularios).
